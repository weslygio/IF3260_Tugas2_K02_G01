How To Use

  1. Choose object (*.json) to load with "Open Design" button.
  2. Turn on/off shader to show/unshow lighting in object.
  3. Use the sliders in "Object" section to change the object's position, rotation, and scale.
  4. Use the sliders in "Camera" section to change camera's position.
  5. Choose projection type to use: perspective, orthographic, and oblique.
  6. Use the sliders in "Light" section to change light's position and color picker to change the light's color.
